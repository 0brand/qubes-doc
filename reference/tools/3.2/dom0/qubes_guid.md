NAME
====
qubes_guid

SYNOPSIS
========
| qubes_guid -d domain_id [-c color] [-l label_index] [-i icon name, no suffix] [-v] [-q]

OPTIONS
=======
-v
    Increase log verbosity
-q
    Decrease log verbosity

Log levels:
    0. only errors
    1. some basic messages (default)
    2. debug
